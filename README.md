# Minimal API - .NET8

Uma api completa feita com o .net8.

Ela possui registro, login e autenticação de usuário. Autorização para endpoints, e a parte do crud para a classe de fornecedores.

### Bibliotecas utilizadas
- EntityFrameworkCore (https://github.com/dotnet/efcore) - para manipulação do banco de dados
- MiniValidation (https://github.com/DamianEdwards/MiniValidation) - para validação das entidades
- NetDevPack.Identity (https://github.com/NetDevPack) - para verificação de usuário

Enjoy :)
